# lesson-2-homework

### Требования
- node.js **v22.16.0**  
- npm **10.9.2**

### Установка зависимостей
```bash
# 1. Убедитесь, что у вас нужная версия Node.js
nvm use 22.16.0 

# 2. Установка всех зависимостей
npm install

# MPA-версия
npm run mpa

# SPA-версия
npm run spa

# SPA-версия со звездочкой
npm run spa-extra
```
